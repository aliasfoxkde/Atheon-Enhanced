Atheon

Scan your code, configs, and environment for leaked secrets before they become a problem.

Download Windows  ·  macOS  ·  Linux  ·  no install

What it is

Atheon is a command-line tool that scans files, directories, environment variables, or piped input and flags any lines that match known secret patterns — API keys, tokens, credentials. It tells you exactly which file, which line, and what was found.

It ships with patterns for the most common providers out of the box, and adding your own takes a single Go file.

atheon ./src
atheon --file config.yaml
atheon --env
cat pipeline.log | atheon -

Example

You're about to push a feature branch. You want to make sure no credentials slipped into the diff.

$ atheon ./src

[aws-access-key] config/deploy.yaml:14  →  AWS_KEY=AKIAIOSFODNN7EXAMPLE
[openai-api-key] .env.local:3           →  OPENAI_API_KEY=sk-proj-abc123...

Two findings, two lines. Fix them before the push. Exit code 1 means something was found — wire that into your CI pipeline and the check runs automatically on every commit.

Built-in patterns

Atheon ships with detection for the most commonly leaked credentials: AWS access keys, GitHub personal access tokens, OpenAI API keys, Slack tokens, Stripe keys, and Twilio tokens. Run atheon list to see everything loaded.

Extend it

The pattern contract is two methods. If you can write a regex or any boolean check, you can add a pattern. Drop the file in patterns/, rebuild, and it shows up in atheon list automatically.

func (p *myPattern) Name() string             { return "my-pattern" }
func (p *myPattern) Matches(line string) bool { return p.re.MatchString(line) }

No config format, no DSL, no plugin manager — just Go code alongside the engine.
